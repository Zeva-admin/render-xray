Инструкция по подключению
  • Установите v2rayNG из Google Play.
  • Скопируйте ссылку подписки кнопкой выше.
  • В приложении: меню → «Группы подписок» → «+» → вставьте ссылку → Сохранить.
  • Три точки → «Обновить подписку», выберите сервер Cocut, нажмите ▶.
  • Установите FoXray или V2Box из App Store.
  • Скопируйте ссылку подписки кнопкой выше.
  • В приложении: Subscriptions → «+» → Import from URL → вставьте ссылку.
  • Обновите конфигурацию, выберите сервер Cocut и нажмите «Connect».
  • Скачайте Hiddify или v2rayN с GitHub.
  • Скопируйте ссылку подписки кнопкой выше.
  • В приложении: Подписка → Настройки подписки → Добавить → вставьте URL.
  • Обновите (Ctrl+F), выберите сервер Cocut и нажмите Enter.
  • Установите Hiddify или V2Box для macOS.
  • Скопируйте ссылку подписки кнопкой выше.
  • Добавить подписку → вставьте URL → Сохранить.
  • Обновите список серверов, выберите Cocut и включите VPN.
